⚠️ スプレッドシートID設定確認

問題: データが保存されない原因

GASコードのスプレッドシートIDが設定されていない可能性が高いです。

以下の手順で確認・修正してください。

ステップ 1: スプレッドシートIDを取得

1. Google スプレッドシートを開く

2. URLを確認：

https://docs.google.com/spreadsheets/d/1234567890abcdefg.../edit

3. ハイライト部分がスプレッドシートIDです

ステップ 2: GASコードを確認

Google Apps Scriptで以下の行を探してください：

const spreadsheetId = 'ここにスプレッドシートIDを入力'; // ← 変更する場所

この行が変更されていない場合、データは保存されません。

ステップ 3: 修正後のコード

以下のように修正してください：

const spreadsheetId = 'YOUR_ID_HERE';

ステップ 4: テスト関数を実行

GASエディタで以下のテスト関数を実行して確認：

function testSpreadsheetConnection() { const spreadsheetId = 'YOUR_SPREADSHEET_ID'; // ← ここに実際のIDを入力 try { const sheet = SpreadsheetApp.openById(spreadsheetId); console.log('✅ スプレッドシートに接続成功！'); console.log('シート名:', sheet.getName()); const targetSheet = sheet.getSheetByName('回答一覧'); if (targetSheet) { console.log('✅ 「回答一覧」シートが見つかりました'); targetSheet.appendRow(['接続テスト', new Date()]); console.log('✅ テストデータを追加しました'); } else { console.log('❌ 「回答一覧」シートが見つかりません'); } } catch (error) { console.log('❌ エラー:', error.toString()); } }
